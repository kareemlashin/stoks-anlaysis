# 📋 استاندرد تحليل الصور

> ملف رفرينس ثابت — أي صورة تتبعت، تتحلل بالكامل حسب الخطوات دي.

---

## 🎯 القاعدة الأساسية
أي صورة = **تحليل شامل كامل**، مش وصف سطحي. متتساب أي تفصيلة.

---

## 🔟 خطوات التحليل (بالترتيب)

### 1️⃣ نظرة عامة
- إيه نوع الصورة؟ (تحليل طبي / مستند / كود / عام…)
- محتواها في سطر أو اتنين.

### 2️⃣ استخراج كامل (OCR)
- قراءة **كل** نص ورقم وعنصر ظاهر في الصورة.
- متسيبش أي بيانات — حتى الصغيرة أو الجانبية.

### 3️⃣ تحليل متخصص حسب النوع

#### 🩺 تحاليل طبية / أشعة
| العنصر | المطلوب |
|--------|---------|
| كل قيمة | الرقم الظاهر |
| المعدل الطبيعي | الرينج الطبيعي لكل تحليل |
| التقييم | طبيعية ✅ / مرتفعة ⬆️ / منخفضة ⬇️ |
| الدلالة | المعنى الطبي المحتمل |

> ⚠️ **تنويه:** التحليل للتوعية فقط ومش بديل عن استشارة الطبيب.

#### 📄 مستندات / سكرين شوت
- استخراج المحتوى كامل.
- تلخيص المعنى.
- تحليل الأرقام / الجداول / السياق.

#### 💻 كود / شاشات برمجة
- تحديد المشكلة أو الخطأ.
- سبب المشكلة.
- الحل المقترح **بالكود**.

#### 🔍 عام / أي حاجة
- تحليل شامل من كل الزوايا: المحتوى، السياق، التفاصيل الملحوظة.

### 4️⃣ ملاحظات وتحذيرات مهمة
- أي حاجة لافتة أو محتاجة انتباه.

### 5️⃣ الخلاصة والتوصية
- المعنى النهائي + الخطوة الجاية المقترحة.
