# 📸 Workflow — صور بس

> **مش محتاج CSV ولا أرقام مكتوبة** — ابعت الصور وأنا أستخرج كل حاجة  
> ⚠️ تحليل تعليمي — مش نصيحة استثمارية

---

## ✅ إيه اللي تبعت (كفاية)

### 📆 متابعة يومية (بعد 14:30)

```
📆 daily
```

+ **صورة يومية (Daily)** لكل سهم في محفظتك (شارت TradingView)

**اختياري:** سطر واحد لو حصل حاجة  
`ملاحظة: AGM بكرة` · `خبر: ...`

---

### 🔬 تحليل عميق / رؤية شاملة

```
📊 deep EAC
```
أو
```
رؤية شاملة
```

+ **صور متعددة الفريمات** — كل ما عندك (شهري · أسبوعي · يومي · 4H · 1H · 15M)

**مش لازم:** CSV · أرقام إغلاق/حجم · ملفات

---

### 📅 فحص أسبوعي (خميس)

```
📅 weekly
```

+ صور **أسبوعي + يومي** للأسهم اللي عايز تراجعها

---

## 🖼️ أفضل طريقة للصورة

| ✅ كويس | ❌ يصعّب القراءة |
|---------|------------------|
| سكرين شوت TradingView كامل | قصّ نص الشارت |
| واضح: السعر · OHLC · الحجم · المؤشرات | ضبابية · zoom قوي |
| **يومي إلزامي** للمتابعة | صورة بدون مقياس |
| فريمات متعددة للتحليل العميق | صورة واحدة بس للعميق |

**مش لازم:** تصدير CSV · Excel · PDF

---

## 🤖 إيه اللي بيحصل لما تبعت صورة

1. **قراءة كاملة (OCR)** — فتح · أعلى · أدنى · إغلاق · حجم · كل المؤشرات
2. **قراءة MTF** — كل فريم لو بعت أكتر من صورة
3. **100 أداة + 15 محترف** — من الصورة + [`محفظتي.md`](محفظتي.md)
4. **PnL · EV · R:R** — على كميتك الفعلية
5. **الحكم** — HOLD / EXIT / ADD + triggers + limit بس

**Python/CSV:** أدوات `_مشترك/tools/` **اختيارية** — مش مطلوبة منك
