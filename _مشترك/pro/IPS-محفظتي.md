## 3️⃣ Entry Rules (ALL required)

- [ ] Investment Thesis completed ([`TEMPLATE-investment-thesis.md`](TEMPLATE-investment-thesis.md))
- [ ] Pre-Mortem completed ([`TEMPLATE-pre-mortem.md`](TEMPLATE-pre-mortem.md))
- [ ] **Trigger hit** — not anticipation
- [ ] R:R ≥ **2:1** to first target
- [ ] EV **positive**
- [ ] Liquidity: exit full position ≤ **3 trading days**
- [ ] Market regime: not 🔴 (distribution ≥ 5 days)
- [ ] Stop defined **before** entry

## 4️⃣ Execution Rules

| Rule | Detail |
|---|---|
| **Order type** | Limit only on microcap / illiquid |
| **Tranches** | 2–3 splits min on size > 50K EGP/day vol |
| **No market blind** | Especially event days (AGM, results) |
| **Timing** | Big decisions: first/last hour (Raschke) |
| **Add to winners** | Only above breakeven + trigger |
| **Never average down** | Without new thesis + pre-mortem |

---

## 5️⃣ Exit Rules

| Trigger | Action |
|---|---|
| Hard stop hit | EXIT 100% — no debate |
| Thesis invalidated | EXIT or reduce 50% same session |
| Time stop (no progress) | Reduce per thesis |
| Target 1 (R=2) | Sell ⅓, trail stop |
| Climax / distribution | Sell into strength |
| O'Neil: 5 dist days on index | No new longs; tighten stops |

---

## 6️⃣ Forbidden (hard no)

- ❌ Market order on microcap with position > 1 day ADV
- ❌ "Buy now" without trigger
- ❌ Trade without stop
- ❌ Revenge trading after loss
- ❌ Override pre-mortem kill switch
- ❌ New long when IPS regime = 🔴
- ❌ Size up on hope before catalyst
