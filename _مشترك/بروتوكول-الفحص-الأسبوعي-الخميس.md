# 📅 بروتوكول الفحص الأسبوعي — **تجهيز شراء كل أسبوع**

> 🎯 **الهدف:** **نشتري إيه الأسبوع ده؟** — مش متابعة يومية  
> 📌 **الموعد:** كل **خميس** بعد إغلاق شمعة Weekly  
> 📊 **المدخلات:** **200 سهم** × **5 فريمات**  
> 📤 **المخرج:** **Buy List** (Top 15) + Trigger + Sizing + Out list  
> ⚠️ تحليل تعليمي — مش نصيحة استثمارية

## ⏰ التوقيت (EGX)

| الحدث | الوقت |
|---|---|
| آخر جلسة في الأسبوع | **الخميس** 10:30–14:30 |
| شمعة Weekly تُقفل | بعد إغلاق الخميس |
| **أنت تبعت الملفات** | الخميس مساءً أو الجمعة |
| **التسليم المتوقع** | خلال 24–48 ساعة (حسب شكل المدخلات) |

**قبل أي فحص:** فحص **حالة السوق** (EGX30/EGX70):
- 🟢 Risk-on → نوسّع قائمة الشراء
- 🟡 Mixed → Top picks بس + triggers صارمة
- 🔴 Distribution (5 أيام توزيع في 25 جلسة — قاعدة أونيل) → **مفيش longs جديدة** إلا استثناءات VCP نادرة

## 📦 شكل المدخلات — **اختار واحد**

### ✅ الطريقة المفضلة: CSV (دقة 100% · أسرع لـ 200 سهم)

```
فحص-YYYY-MM-DD/
├── _market/
│   └── EGX30.csv          # اختياري — للقوة النسبية RS
├── symbols.txt            # 200 كود — سطر لكل سهم
└── data/
    ├── COMI_M.csv
    ├── COMI_W.csv
    ├── COMI_D.csv
    ├── COMI_4H.csv
    ├── COMI_1H.csv
    ├── EAC_M.csv
    └── ...
```

**أعمدة CSV (TradingView Export):**
`time, open, high, low, close, volume`

**تصدير من TradingView:**
1. افتح الشارت → الفريم → ⋮ → **Export chart data**
2. كرّر لكل فريم × كل سهم (أو استخدم screener + export batch لو متاح)

**أو:** ملف CSV واحد مجمّع:
```
symbol,timeframe,time,open,high,low,close,volume
COMI,W,2026-07-03,45.2,46.1,44.8,45.9,1200000
```

## 🏆 نظام الترتيب (Score — 100 نقطة)

### A. الاتجاه — 25 نقطة

| المعيار | W | D | M | النقاط |
|---|---|---|---|---|
| Close > MA20 | ✓ | ✓ | — | +5 |
| MA20 > MA50 (W) | ✓ | — | — | +5 |
| ADX(14) > 25 و +DI > −DI | ✓ | — | — | +5 |
| Weinstein Stage 2 (فوق MA30W صاعد) | ✓ | — | ✓ | +5 |
| Higher Highs + Higher Lows (آخر 8–12 أسبوع) | ✓ | — | — | +5 |

### B. الزخم — 20 نقطة

| المعيار | الفريم | النقاط |
|---|---|---|
| RSI(14) 50–70 (قوة مش تشبع) | D | +5 |
| MACD hist > 0 و صاعد | D | +5 |
| WaveTrend proxy: WT1 > WT2 | D | +5 |
| Connors RSI-2 setup (RSI2<10 فوق MA200) — mean rev | D | +5 * |

\* *bonus lane — لو فلتر mean-reversion*

### C. البنية / SMC — 20 نقطة

| المعيار | الفريم | النقاط |
|---|---|---|
| BOS صاعد (قمة سابقة انكسر close) | W أو D | +7 |
| فوق Premium 50% (آخر swing range) | D | +5 |
| FVG/OB retest نظيف (unmitigated) | 4H | +5 |
| مفيش CHoCH هابط حديث (< 3 أسابيع) | W | +3 |

### D. التقلب / Setup — 15 نقطة

|---|---|
| TTM Squeeze ON → Fire (BBW expansion + close direction) | +8 |
| VCP: 2–4 contractions + vol جاف | +7 |
| ATR% معقول (مش climax > 2× median) | +0 (فلتر سلبي) |

### E. الحجم — 10 نقطة

| المعيار | النقاط |
|---|---|
| Vol أسبوعي > متوسط 10 أسابيع | +5 |
| OBV slope صاعد 8 أسابيع | +5 |

### F. القوة النسبية RS — 10 نقطة

|---|---|

### 3️⃣ Tier Lists

| Tier | المعنى | Action |
|---|---|---|
| **A** (Score ≥ 75) | confluence قوي — 3+ طبقات متفقة | راقب Trigger · حجم كامل حسب sizing |

| **C** (45–59) | marginal | مراقبة بس · مفيش دخول |
| **OUT** (< 45 أو فلتر) | rejected | تجنّب |

```

```
