### 📐 قواعد محاكاة SMC موحّدة (LuxAlgo / ChartPrime / BigBeluga / Zeiierman)

```
1. Order Block     → آخر شمعة معاكسة قبل BOS/اندفاعة (high volume)
2. FVG             → gap بين high[i-2] و low[i] (bull) أو العكس
3. BOS/CHoCH       → close فوق/تحت swing high/low مؤكد
4. Liquidity Pool  → equal highs/lows (2+ touches same level ±0.5%)
5. Premium/Discount→ 50% of range (from swing low to high)
6. OTE             → 62–79% fib retracement of impulse
7. Smart Trail     → ATR-based trailing stop (LuxAlgo ≈ 2.5×ATR from swing)
8. Confluence      → count independent signals at same level (0–5)
```
